JetBrains IDE Shortcuts

If you learn only one


 + A
Search for an editor command

Navigation


 + O
Open a file by name (uses fuzzy matching!)

+ [ / ]
Navigate the cursor history

+ [ / ]
Switch editor tabs

+ B
Jump to the declaration of the selected item

Running Tests


 + R
Run the test containing the insertion point

+ R
Re-run the most recent test(s)

Editing Code


 + enter
Accept a suggestion (when the lightbulb appears)

Git


 + K
View the commit dialog

Search and Replace


 + F
Find in current file

+ R
Replace in current file

+ F/R
Find/replace globally

+ G
Multi-select
